# 财务 M7 定向重放报告

## 结论

本报告使用上一轮 120 个真实网页节点的完整账本快照作为输入，逐个调用当前版本的开局事实摄取和结算前完整性政策；它验证政策能否拦截旧数据中的已知缺口，不把旧结果冒充新网页结果。

| 指标 | 结果 |
|---|---:|
| 路线 | 5 |
| 节点 | 120 |
| 明确房贷开局 / 新入口成功摄取 | 1 / 1 |
| 成年支出缺失节点 / 政策失败 | 22 / 0 |
| 晚年工资过期节点 / 政策失败 | 34 / 0 |

重放结论：**通过**。

注意：重放只能证明旧缺口会被新政策捕获；Prompt 的真实输出质量仍由三组短网页测试验证。
